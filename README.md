## 实验一 哈夫曼图片压缩
1. 压缩
    1. 输入bmp图片的路径
    2. 生成文件会在原名称基础上加.huf，路径一致
2. 解压
    1. 输入解压huf文件的路径
    2. 输入生成文件路径（以.bmp为后缀）
**注意输入路径时斜杆的正确输入**  
**若/是单斜杠/，\则是双斜杠\\**
